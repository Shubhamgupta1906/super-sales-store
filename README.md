# 📊 Super Store Sales Dashboard (Power BI + Power Query)

An interactive **Power BI dashboard** built with **Power Query** for data transformation and modeling, enhanced using **DAX measures** and **calculated columns**.  
The dashboard connects to a **Google Sheets data source** for real-time updates and analysis.

---

## 🚀 Features

- **Dynamic KPIs:** Orders, Sales, Profit, Shipment Days  
- **Regional Filters:** Central, East, South, West  
- **Interactive Visualizations:**
  - Sales by Category (Consumer, Corporate, Home Office)
  - Sales by Payment Method (Cards, COD, Online)
  - Monthly Sales & Profit Trends (2019–2020)
  - Sub-category Performance (Phones, Chairs, Binders)
  - Shipment Type Breakdown (Standard, Second Class, First Class, Same Day)
  - Sales & Profit by State (Map Visualization)
- **Data Source Integration:** Connected directly to **Google Sheets** for live data refresh  
- **Data Modeling:** Power Query transformations + DAX measures and calculated columns  

---

## ✨ Highlights

- Built entirely in **Power BI Desktop** with **Power Query** for ETL  
- Uses **Google Sheets** as a cloud-based data source for easy collaboration and updates  
- Custom **DAX measures** for KPIs like Total Sales, Profit Margin, Orders Count, and YoY comparisons  
- **Calculated columns** for advanced segmentation (shipment types, categories)  
- Clean, professional layout designed for **business decision-making**  
- Provides actionable insights for sales, profit, and logistics optimization  

---

## 📊 Why These Charts?

Each chart was chosen to answer specific business questions:

- **Sales by Category** → Understand which customer segments drive revenue.  
- **Sales by Payment Method** → Identify preferred payment channels to optimize checkout.  
- **Monthly Sales & Profit Trends** → Spot seasonal patterns and profitability fluctuations.  
- **Sub-category Performance** → Highlight top-selling products for inventory planning.  
- **Shipment Type Breakdown** → Evaluate logistics efficiency and delivery preferences.  
- **Sales & Profit by State (Map)** → Provide geographic insights for marketing and regional strategies.  
- **Top KPIs (Orders, Sales, Profit, Ship Days)** → Quick snapshot of overall performance.  

---

## 🛠️ Tech Stack

- **Power BI Desktop** – dashboard creation and visualization  
- **Power Query** – data cleaning, transformation, and modeling  
- **DAX Measures & Calculated Columns** – advanced KPIs and logic  
- **Data Source:** Google Sheets  

---

## 🔗 Data Source

- **Google Sheets**: The Super Store dataset is stored in Google Sheets.  
- Connected directly to **Power BI** using the Google Sheets connector.  
- **Power Query** is used to:
  - Remove nulls and duplicates  
  - Merge tables and shape data  
  - Create calculated columns for categories and shipment types  
- **DAX Measures** are applied for:
  - Total Sales  
  - Profit Margin  
  - Orders Count  
  - Year-over-Year comparisons
